colophon

This site is powered by Hugo, the static-site generator.

It runs a heavily modified version of the Archie theme. I started with Archie because it’s minimal and Markdown-inspired, but I have since changed it to match my tastes and uses.

I seriously considered going full motherfu*kingwebsite.com: just raw HTML and CSS, no build step. It’s still tempting, but Hugo is stupidly fast, the templating is delightful, and I get to write everything in Markdown. So here we are.

Everything is written in my editor of choice, Neovim.

Technical details

  • Built with hugo --minify --gc
  • Zero JavaScript
  • Monospace system stack only: Menlo, Monaco, "Courier New", Consolas, "Liberation Mono", "Lucida Console", monospace
  • Colors from the Everforest theme
  • Single CSS file
  • Images are .avif, compressed with ShortPixel
  • No web fonts, no cookies, no analytics, no trackers, no telemetry

It loads instantly and wastes nothing.

Built with love, stubbornness, and the intent to keep things small.
