colophon

This website is written and maintained by Michael Oliveira.

It is powered by Hugo, the static-site generator.

Initially, I considered going full motherfu*kingwebsite.com with just raw HTML and CSS. It is still tempting, but Hugo is extremely fast, the templating is delightful, and I get to write everything in Markdown, so here we are.

All text is written in Markdown using my editor of choice, Neovim (check out my .config!).

Text is set with a system-ui font stack: system-ui, sans-serif.

Social icons are from feathericons.com, and the 88x31 buttons come from cyber.dabamos.de.

Dates are formatted using the Arvelie date system.

This site uses no tracking, analytics, or cookies.

Check out this site in w3m or lynx!
